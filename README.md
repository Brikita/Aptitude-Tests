# GIA Aptitude Master

A web simulator for the Thomas General Intelligence Assessment (GIA) consisting of 5 distinct cognitive tests.

## Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Tasks

1.  **Reasoning**: Compare two people based on a statement.
2.  **Perceptual Speed**: Identify matching pairs in a set of 4.
3.  **Number Speed & Accuracy**: Find the number furthest from the median.
4.  **Word Meaning**: Identify the word that doesn't fit.
5.  **Spatial Visualisation**: Mental rotation and matching.

## Getting Started

1.  Install dependencies:

    ```bash
    npm install
    ```

2.  Run the development server:

    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser.
